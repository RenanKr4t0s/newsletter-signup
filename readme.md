# BACKEND-NewsLetter


## Sobre:

Repositório Backend com Express.js conectado com API do Mailchimp para treinar comunicações externas, get, post e mais.
  
## Tecnologias Utilizadas:

- BodyParser
- Express.js
- Mailchimp API
- Bootstrap
